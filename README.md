Инструкция по использованию гитхаб в интелке:
1) пкм по файлы или папке Git -> Add
2) Main menu -> Git -> Commit и написать текст любой
3) Main menu -> Git -> Push и выбрать текст который ты писал в пункте 2

4) Main menu -> Git -> Pull это «принять» изменения на своей стороне

Объяснение:
1) пункты 1-3 добавление файла на гитхаб
2) пункт 4 чтобы «принять» изменения (ну если ты тупой то можешь переустановить проект чтобы принять изменения)
